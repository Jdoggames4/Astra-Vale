# 🧩 *Features*
## ⌨️ **Core Gamplay**
  - **Smooth Movment**: Running, jumping, double jumping, and wall jumping
  - **Physics Movment**: Realitic gravity and collision
  - **Charecter Animations**: Detailed Animations

## 🔥 *Game Elements*
  - **Terrain**: Various platofrm types
  - **Traps**
    o 🔥 Fire traps
    o ⚙️ Rotating saws
    o 📌 Spikes
  - **Sound Effects**: Background music and sound effects

## 🎮 *Controls*
  - **Keyboard Support**: Arrow keys for movment
  - **Menu System**: A beautiful animated main menu

## 🎨 *Visual Features*
  - **High-Quality Assets: Hand crafted by ukihunter
  - **Smooth Animations**: Fluid charecter movement and object interation
  - **Partical Effects**: Enhanced visuals

# 🚀 Installation

 - Python 3.6 or higher
 - Pygame libaray

# 🎮 Controls
## Keyboard Controls
  - **Left/Right Arrow**: Move left/right
  - **Space**: Jump

# 🎯 Game Mechanics
## Trap Types
  1. **Fire Traps**: Animated fire obstacles
  2. **Saws**: Rotating circlar blades
  3. **Spikes**: Spike traps
